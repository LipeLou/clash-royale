# Clash Royale

## Resumo
Ferramenta em desenvolvimento para acompanhar, em tempo real, a mão e o deck do oponente no Clash Royale durante partidas no modo espectador.

## Objetivo
Identificar as 8 cartas do oponente, rastrear a ordem do baralho (lógica FIFO) e estimar o elixir disponível enquanto a partida acontece.

## Ferramentas
- Python
- OpenCV
- mss
